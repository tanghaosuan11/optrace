# Third-Party Licenses

This project uses open-source dependencies from both JavaScript/TypeScript and Rust ecosystems.

Generated reports:

- Frontend (npm/pnpm), JSON: `licenses/frontend-licenses.json`
- Frontend (npm/pnpm), Markdown: `licenses/frontend-licenses.md`
- Rust crates, JSON: `licenses/rust-licenses.json`
- Rust crates, TSV: `licenses/rust-licenses.tsv`

Notes:

- These files are generated from the current lockfiles and dependency graph.
- Regenerate them after dependency updates to keep license information current.
